# 🛒 Blinkit Sales Dashboard (Excel Analytics Project)

## 📌 Project Overview
This project showcases an interactive Excel dashboard built using Blinkit grocery sales data. The dashboard helps analyze sales performance, customer preferences, and outlet distribution using data visualization techniques.

---

## 🎯 Objectives
- Analyze total sales and average performance
- Understand customer preferences (fat content, item types)
- Compare outlet performance across locations and sizes
- Identify trends over years

---

## 📊 Key Performance Indicators (KPIs)

| KPI | Value |
|-----|------|
| Total Sales | $0.09M |
| Average Sales | $140 |
| Total Items | 649 |
| Average Rating | 4.0 |

---

## 📈 Dashboard Features

### 🔹 Sales Analysis
- Total sales by outlet type
- Year-wise outlet establishment trend

### 🔹 Product Insights
- Sales by item type
- Fat content distribution (Low Fat vs Regular)

### 🔹 Outlet Insights
- Sales by outlet size (Small, Medium, High)
- Sales by outlet location (Tier 1, Tier 2, Tier 3)
- Outlet type comparison

### 🔹 Interactive Filters
- Outlet Size
- Outlet Location Type
- Item Type

---

## 🛠 Tools & Techniques Used

- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers (Filters)
  - KPI Cards
  - Conditional Formatting
  - Data Cleaning

## 📂 Dataset Description

## 🔍 Key Insights

- Tier 3 outlets generate the highest sales
- Medium-sized outlets dominate sales contribution
- Regular fat products slightly outperform low-fat items
- Supermarket Type 1 has highest number of items
